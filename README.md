## Minting page scaffold

Pagina utilizada para mintar colecciones en noajpg. 

## Tecnologias

``` 
  Reactjs
  Nextjs
  Typescript
```

## Correr el proyecto

Instalar dependencias con el comando 

``` bash
 $ npm i 
```

Correr el proyecto con el comando 

``` bash
 $ npm run dev
```
